อธิบายควาหมมายของระบบฝังตัว 
-ตอบ- คอมพิวเตอร์ที่ถูกฝังหรือติดตั้งไว้ในสิ่งต่างๆ เพื่อให้สิ่งนั้นทำงานได้ตามที่ต้องการโดยทั่วไปจะเป็นงานเฉพาะด้าน เช่น ทางการแพทย์
ความแตกต่างระหว่าง computer system, non-computerized system
-ตอบ- non computerized system เป็นระบบที่ไม่ใช่คอมพิวเตอร์เลยใช้เป็น manual ส่วน computerized system จะมีคอมพิวเตอร์เป้นส่วนหนึ่งของระบบ
ประเภทของระบบฝังตัว
-ตอบ- real-time, standalone, networked, mobile, embedded subsystem
ความแตกต่างระหว่าง Soft real-time embedded systems และ hard real-time embeddedsystems
-ตอบ- soft จะอนุญาติให้มี delay เล็กน้อยแต่ hard จะไม่อนุญาติให้มี delay เพราะต้องมีความแม่นยำในการประมวลผล เช่น ระบบรถ auto
ความแตกต่างระหว่างระบบฝังตัวกับคอมพิวเตอร์แบบทั่วไป
-ตอบ- คอมพิวเตอร์สามารถใช้ได้หลายอย่างมากกว่าในขณะที่ระบบฝังตัวมีไว้เพื่อวัตถุประสงค์เดียวเท่านั้น
ยกตัวอย่างผลิตภัณฑ์ที่มีระบบฝังตัวติดตั้ง
-ตอบ- แอร์ ไมโครเวฟ
อธิบายภาพรวมของระบบฝังตัว
-ตอบ- ปกติค่าทำหน้าที่รับค่าและส่งต้องค่าที่ได้ เซ็นเซอร์จะตรวจสอบและส่งค่าไป processor ประมวลผล input หลังจากนั้นจะตัดสินใจว่าจะทำยังไงไปยัง output
อธิบายหน้าที่ของไมโครคอนโทรลเลอร์
-ตอบ- ควบคุมระบบมีขนาดเล็กมีลักษณะคล้าย mainbroad ของ PC แต่มีขนาดเล็กกว่า
อธิบายความแตกต่างระหว่าง input และ output 
-ตอบ- input เป็นข้อมูลที่ภายนอกส่งเข้ามาในระบบแต่ output เป็นข้อมูลที่ออกไปทำงานหน้าที่ต่างๆ เช่น ...
อธิบายความแตกต่างระหว่างสัญญาณ analog และ digital 
-ตอบ- analog จะแปรผันตามแรงดันไฟฟ้าค่าตัวเลขจะมีทศนิยมส่วน digital จะมีแค่ 2 ค่าคือ 0,1 
อธิบายประโยชน์ของ automation 
-ตอบ- เพิ่มผลผลิต, เพิ่มคุณภาพให้งาน, ลดค่าใช้จ่ายหรือต้นทุน
ยกตัวอย่างการใช้งานของเซ็นเซอร์ประเภทต่างๆได้
-ตอบ- เซ็นเซอร์อุณหภูมิ, ระยะทาง, ความร้อน, แรงดัน
บอกความแตกต่างระหว่างเซ็นเซอร์แบบ analog และแบบ digital ได้
-ตอบ- analog จะอ่านค่าอย่างเดียมีการทำงานในตัวน้อยเป็นสัญญาณต่อเนื่อง ส่วน digital จะต้องแปลงค่าจาก analog มาเป็น digital ก่อนให้อยู่ในรูป 0,1 เป็นสัญญาณไม่ต่อเนื่อง
บอกความหมายและยกตัวอย่างการทำงานของตัวกระตุ้นได้และยกตัวอย่างการใช้งานอุปกรณ์ต่อพ่วงอื่นๆได้
-ตอบ- เป็นสิ่งที่กระตุ้นให้บางสิ่งทำงาน เช่น มอเตอร์ที่จะทำให้อุปกรณ์มีการหมุนตัวเปิดปิดวาร์ดได้
อธิบายหลักการทำงานของ RFID ได้
-ตอบ- RFID จะอ่านข้อมูลจากเครื่องสแกนแต่จะมีความยืดหยุ่นมากกว่าบาร์โค้ดเพราะมีการซ่อนแท็ก
เข้าใจความหมายและประโยชน์ของ Internet of Things
-ตอบ- คือ อินเทอร์เน็ตในสรรพสิ่ง หมายถึง เครือข่ายรวมของอุปกรณ์ที่เชื่อมต่อถึงกันและเทคโนโลยีที่อำนวยความสะดวกในการสื่อสารระหว่างอุปกรณ์กับระบบคลาวด์ ตลอดจนระหว่างอุปกรณ์ด้วยกันเอง ทำให้เราเข้าถึงข้อมูลได้ง่าย, ติดต่อสื่อสารดีขึ้น, หลายสิ่งทำงานแบบ auto ได้, ประหยัดเวลา
บอกการประยุกต์ใช้งาน Internet of Things
-ตอบ- ใช้บ้าน เช่น smart home เปิดปิดผ่านโทรศัพท์หรือ smart farm ที่ตรวจสุขภาพของสัตว์
เข้าใจความหมายของ Wireless Sensor Network
-ตอบ- เครือข่ายของเซ็นเซอร์ที่มีการติดต่อสื่อสารกันผ่านสัญญาณวิทยุที่มีขนาดเล็กอาจติดอยู่ในสิ่งของเพื่อตรวจสอบอุณหภูมิ
